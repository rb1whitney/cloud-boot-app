# Product Definition: Programming-Work

## Vision
A centralized, high-performance library and development hub for specialized AI agents and skills.

## Goals
- Provide a single source of truth for across multiple LLM tools (Claude, Gemini, Copilot).
- Enable rapid development and testing of automated coding and research skills.
- Maintain a premium, well-documented repository of AI capabilities.

## Managed Applications

### Cloud Boot App
A modernized Java 21 / Spring Boot 3.2 backend application serving as a reference implementation for:
- **Containerization**: Secure Google Distroless runtime.
- **Documentation**: Automated SpringDoc OpenAPI integration.
- **Modern Infrastructure**: Deployable via Terraform and Jenkins.
